# HKUAA Asset Slots

Put your approved visual assets here.  
The current frontend is already wired to these paths:

- `public/assets/home/hero.jpg`
- `public/assets/home/about.jpg`
- `public/assets/home/clubhouse.jpg`
- `public/assets/home/membership.jpg`

Recommended minimum sizes:

- hero: 1920x1080
- section cards: 1200x800

If files are missing, the site falls back to gradient placeholders.
